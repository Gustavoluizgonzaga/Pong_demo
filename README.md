# Jogo Pong – Projeto de Lógica - Feito no curso CS50

Projeto desenvolvido para praticar lógica de programação por meio da criação de um jogo simples inspirado no clássico Pong.

O foco do projeto é o controle de estados, movimentação de objetos e interação com o usuário.

## Tecnologias Utilizadas
- Lua

## Funcionalidades
- Movimento de jogadores
- Controle de colisões
- Sistema simples de pontuação

## Objetivo do Projeto
Exercitar lógica de programação, estruturação de código e desenvolvimento de aplicações interativas.

## Aprendizados
- Controle de fluxo e estados
- Estruturação de lógica para jogos
- Organização de código
